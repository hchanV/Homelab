# Priority 0 — Firmware / BIOS

## Goals

- Update BIOS if newer stable version exists
- Confirm proxmox stability post update

## Why this matters

BIOS updates may improve:

- NVMe compatibility
- Memory compatibility
- PCIe stability
- Virtualization support
- Power management
- Wake-on-LAN behavior

## Notes

- BIOS updates sometimes reset:
  - VT-x / AMD-V
  - IOMMU
  - Boot order
  - Secure Boot
- Verify Proxmox still boots after updating

## SSH Hardening

- Verify SSH is needed
- Disable password SSH login later (after keys are configured)
- Generate SSH keypair on main workstation
- Add public key to Proxmox

Commands:

```bash
systemctl status ssh
```

Later review:

```bash
nano /etc/ssh/sshd_config
```

Settings to review later:

```text
PermitRootLogin prohibit-password
PasswordAuthentication no
```

⚠️ Do not disable password login until SSH keys are tested.
