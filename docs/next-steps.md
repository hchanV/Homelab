# Homelab first steps!

## Current Goal
Build a home lab setup for learning Linux, storage, and various other self-hosted services

## Planned Services
- File server
- Print server
- Media server
- Security camera system
- Self-hosted password manager
- Other useful services to be determined

## Current Decision
I am currently considering Proxmox for the central server which will be henceforth known as Tyranitar, with ZFS for storage and separate VMs/containers for services.
The reason being is we want a central server that can handle a number of different other containers/vm's
